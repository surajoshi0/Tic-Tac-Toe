## About

This Tic Tac Toe game is a fun, interactive game playable directly in the browser. The game is for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The first player to get three of their marks in a row (vertically, horizontally, or diagonally) wins the game.

## Features

- Simple and intuitive UI
- Two-player game (Player X and Player O)
- Displays the winner or a draw message
- Option to restart the game
